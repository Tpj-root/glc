### ‚öôÔ∏è Basic Usage

```bash
nmap [options] [target]
```

#### **Examples**

* Scan a single IP:

  ```bash
  nmap 192.168.1.1
  ```

* Scan a range:

  ```bash
  nmap 192.168.1.1-10
  ```

* Scan subnet:

  ```bash
  nmap 192.168.1.0/24
  ```

* Scan hostname:

  ```bash
  nmap example.com
  ```

* Scan specific ports:

  ```bash
  nmap -p 22,80,443 192.168.1.1
  ```

* Aggressive scan (OS detect, version, script):

  ```bash
  nmap -A 192.168.1.1
  ```

* Stealth scan (SYN scan):

  ```bash
  nmap -sS 192.168.1.1
  ```

* Service and version detection:

  ```bash
  nmap -sV 192.168.1.1
  ```
